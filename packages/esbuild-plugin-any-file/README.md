## Install

```sh
npm i @chialab/esbuild-plugin-any-file -D
```

```sh
yarn add @chialab/esbuild-plugin-any-file -D
```

## Usage

```js
import esbuild from 'esbuild';
import filePlugin from '@chialab/esbuild-plugin-any-file';

await esbuild.build({
    plugins: [
        filePlugin(),
    ],
});
```
